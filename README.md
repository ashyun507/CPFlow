# CPFlow

This repository contains the official implementation of **CPFlow**, the open-source codebase for the paper:

**CPFlow: Receptor-Conditioned Cyclic Peptide Generation with Autoregressive Flow Matching**

CPFlow is a receptor-conditioned generative framework for cyclic peptide sequence-structure co-design.

## Overview

The code will include model training, inference, and evaluation scripts for CPFlow.

## Status

The repository is being organized for release.
